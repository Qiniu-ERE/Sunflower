## 🚀 部署升级

### 从v2.2升级到v3.0.0

#### 1. 备份数据

```bash
# 备份项目数据
cp -r data data.backup
cp -r output output.backup
```

#### 2. 更新代码

```bash
# 拉取最新代码
git pull origin main

# 切换到v3.0.0标签
git checkout v3.0.0
```

#### 3. 更新依赖

```bash
# 更新Python依赖
pip3 install -r requirements.txt --upgrade
```

#### 4. 重启服务

```bash
# 停止现有服务
pkill -f run_web.py

# 启动新版本
python3 run_web.py
```
